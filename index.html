Breve Introducción

Aqui trataremos sobre qué irá la próxima lección, un poco de hisotoria y si la tuviese alguna curiosidad, todo a modo de pequeña introducción

Explicación teorica

Esta es una de las partes más importantes, en la cual se deberá mostrar con la suficiente claridad y destreza la parte teórica del tema a tratar. Todo ello va dirigido a personas que no sabe programar y por tanto les costará más entender todo lo relacionado con la lógica abstracta que supone el aprendizaje de un lenguaje de programación. La dificultad aumenta al ser Java por tener las características de un tipado fuerte y ser muy estricto. Por ello para ofrecer información veraz se constrastará con diversas fuentes, siempre que sean confiables

Ejemplos de usos

Para facilitar la comprensión o resolver la clásica duda de para que usarlo, se mostrarán una serie de situaciónes en las que requieran usar las técnicas aprendidas durante la lección actual. Se mostrarán casos reales para que el alumno enlace la información con la tarea para la que está especificamente diseñada

Problema resuelto

Junto con la teoría, esta parte es de las más criticas del articulo. Con la ayuda del scipt Highlight.js mostraremos al usuario ejemplos de código, todos ellos detallados en cuanto a comentarios incluyendo información de que hace cada parte del programa. La ventaja de este script es que nos evitará hacer capturas y subirlas, con el consecuente peso innecesario para el servidor. Otra de las ventajas es que es muy llamativo en cuando a colores, como si fuera un IDE. Una prueba de este script: 


                        String z = "Hola illo, esto es una prueba de Highlight.js";
                        int a = 12 + 12 * 21;
                        System.out.print("Valor del numero a: " + a);
