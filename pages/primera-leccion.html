ArrayList

En esta primera lección, abordaremos uno de los tipos de Colecciones más utilizados: el ArrayList. Dada su versatilidad y familiaridad, se considera el tipo de colección estándar o predeterminado al igual que se utiliza el tipo de dato primitivo int para declarar un número entero

Características principales

El ArrayList es un "tipo" de array mejorado en el que no es necesario definir un tamaño y puede variar de valor en tiempo de ejecución. Tiene una serie de métodos con los que acceder, modificar o borrar elementos de una manera sencilla y dinámica. Al igual que en todo, Java lo considera como un objeto por lo que hay que tratarlo como tal. Esta lista pertenece a un tipo de las muchas que hay en Java: Linkedlist, Set, Maps, Queues, etc.

La principal ventaja que pueda tener esta lista frente a otras es que podemos acceder a cualquier elemento mendiante su indice de posicion, en los Sets por ejemplo no se puede ya que solo itera de inicio a fin o al revés. Además de garantizar el orden de inserción, podemos redimensionar mediante añadido o borrado de elementos de forma dinámica. Pero no todo es bueno, dependiendo de para que lo utilicemos podemos optar por otro tipo más eficiente.

No añadiremos más teoria por el momento, aunque esta lección de para mucho más

Esto es un problema típico en las aplicaciones web, y para ello podemos apoyarnos en los ArrayList. Otro ejemplo fácil de ver es para un carrito de la compra, al igual que en el ejemplo anterior necesitamos un array que este sujeto a cambios en tiempo real

Porblema resuelto

Ahora viene la parte más divertida: Crear la magia. En este ejemplo usamos una lista para que al sumar dos numeros pertenecientes a un array, nos de el valor de un elemento target. Este es el primer problema que se presenta en la página LeetCode 


            public static int[] twoSumS(int[] numbers, int target) {
                List<Integer> lista = Arrays.asList(Arrays.stream(numbers).boxed().toArray(Integer[]::new));
                int[] index = new int[2];

                for (int i = 0; i < numbers.length; i++) {
                    for (int j = 0; j < numbers.length; j++) {
                        if ((i != j) && (numbers[i] + numbers[j] == target)) {
                            index[0] = lista.indexOf(numbers[i]);
                            index[1] = lista.indexOf(numbers[j]);
                            break;
                        }
                    }
                }
                Arrays.sort(index);
                return index;
            }

Tu turno

Es hora de practicar todo lo aprendido en esta lección. El uso de los ArrayList puede ser poderoso pero como todo hay que usarlo con precaución: Puede no ser la lista mas eficiente dependiendo de la tarea a realizar. A continuación se ofrece un ejercicio propuesto:

  • Crea un porgrama que pida por pantalla los elementos que se van a añadir y su precio
  • Deberá preguntar al usuario cada vez que termine de añadir si quiere finalizar la compra
  • También le preguntará si quiere borrar algún artículo, y en caso afirmativo borrarlo
  • Al final se imprimirá una lista con todos los articulos, sus precios correspondientes, el número total de articulos y precio final con su deduccion de IVA correspondiente

  • Pista: Prueba a crear los artículos como objetos, donde la cantidad de cada articulo y su precio unitario puedes definirlo como atributo de dicho objeto
